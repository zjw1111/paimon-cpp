Detailed Description

Represents a Top-K global index result that combines a Roaring bitmap of candidate row ids with an array of associated relevance scores.

Important Ordering Note: Despite inheriting from TopKGlobalIndexResult, the results are NOT sorted by score. Instead, both the bitmap and the score vector are ordered by ascending row id. This design enables efficient merging and set operations while preserving row id-to-score mapping.
