Detailed Description

RecordBatch encapsulates a batch of data with the same schema, supporting different types such as INSERT, UPDATE_BEFORE, UPDATE_AFTER, and DELETE.

It is typically used in streaming write or batch processing scenarios, with underlying data stored in the Apache Arrow format.

Note
Do not use this class directly, use RecordBatchBuilder to build a RecordBatch which has input validation.
