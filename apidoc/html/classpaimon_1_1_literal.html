Detailed Description

Literal represents a constant value used in predicate expressions.

Literal support BOOLEAN, TINYINT, SMALLINT, INT, BIGINT, FLOAT, DOUBLE, STRING, BINARY, TIMESTAMP, DECIMAL, DATE.
