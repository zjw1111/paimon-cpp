◆ Exclude()

std::vector< Range > paimon::Range::Exclude ( const std::vector< Range > &  ranges) const

Excludes the given ranges from this range and returns the remaining ranges.

For example, if this range is [0, 10000] and ranges to exclude are [1000, 2000], [3000, 4000], [5000, 6000], then the result is [0, 999], [2001, 2999], [4001, 4999], [6001, 10000].
